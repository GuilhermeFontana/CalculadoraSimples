# Calculadora Simples - Java

Aplicacao desenvolvida em Java que executa operacoes matematicas e utilitarios matematicos diretamente pelo terminal.

---

## Sobre o Projeto

Este projeto implementa um menu interativo no console, utilizando estruturas fundamentais da linguagem Java como `Scanner`, `switch-case` e validacao de entrada. O objetivo e demonstrar o uso de logica de controle de fluxo e interacao com o usuario atraves do console.

---

## Funcionalidades

- Calculadora simples (adicao, subtracao, multiplicacao e divisao com validacao por zero)
- Fatorial de um numero (calculo iterativo)
- Tabuada completa do 0 ao 10
- Menu interativo com opcao de saida

---

## Tecnologias

- Java (JDK 8+)
- Nenhuma dependencia externa

---

## Como Executar

1. Clone o repositorio
   ```
   git clone https://github.com/GuilhermeFontana/CalculadoraSimples.git
   ```

2. Compile o arquivo
   ```
   javac Main.java
   ```

3. Execute o programa
   ```
   java Main
   ```
